# Mundial 2026 — Cuadro de cruces

Cuadro interactivo de la fase eliminatoria del Mundial 2026 (Dieciseisavos a la Final).

- Los 16 cruces de Dieciseisavos están confirmados (datos al cierre de la fase de grupos, 27/06/2026).
- Toca un partido, mete el marcador y el ganador avanza solo a la siguiente ronda.
- Los resultados se guardan en el navegador de cada dispositivo (`localStorage`). Usa los botones "Copiar respaldo" / "Cargar respaldo" para pasar tus resultados a otro dispositivo o como copia de seguridad.

## Publicarlo en GitHub Pages

1. Sube `index.html` a la raíz del repo (rama `main`).
2. Repo → **Settings → Pages** → Source: `Deploy from a branch` → rama `main`, carpeta `/ (root)` → **Save**.
3. Espera 1-2 minutos. Tu URL será:
   `https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`

## Actualizar el cuadro más adelante

Cuando quieras los cruces reales de Octavos en adelante (o cualquier corrección), pídeselo a Claude ("actualiza el Mundial") y sustituye este `index.html` por el nuevo en el repo (Add file → Upload files, sobrescribiendo).
